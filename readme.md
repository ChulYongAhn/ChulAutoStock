프로젝트 개요
목표: 한국투자증권 API를 활용한 주식시장 감시 프로그램 → 최종적으로 24시간 서버에서 자동 매수/매도
개발자 배경: Unity + C# 전문, 파이썬 & 서버 경험 없음. 이 프로젝트를 통해 파이썬과 서버 구축을 동시에 학습.
개발 환경:

언어: Python
OS: macOS (맥북)
IDE: VSCode
AI 도구: Claude Code
버전 관리: GitHub + GitHub Desktop


마일스톤
1단계: 로컬 PC에서 안정적으로 동작하는 프로그램 완성
2단계: 24시간 돌아가는 서버에 배포 (클라우드 or 홈서버)
3단계: 매수/매도 기능 추가

1단계 상세 — Phase 구성
Phase 0 — 앱 시작 (프로그램 실행 시)
한국투자증권 API로 로그인 → 접근 토큰 발급. 토큰은 24시간 유효하므로 하루 1회만 발급.
Phase 1 — 과거 탐색 모드 (08:30 ~ 08:50)
사전 지정된 코스피 100개 종목의 전일 데이터(종가, 저가, 고가 등)를 조회하여 캐싱.
Phase 2 — 오늘 데이터 탐색 (반복 실행)
100개 종목의 현재가를 실시간 조회. 전일 종가 대비 등락률 계산 후 +2% ~ +4% 범위 종목만 필터링하여 상위 10개 추출.
Phase 3 — 스코어링 & 순위화 (08:59:00 ~ 08:59:50)
Phase 2에서 추린 10개 중 상위 3개를 선별. 가격, 등락률, 거래량 등 상세 로그 기록.
실행 흐름: Phase 0 → Phase 1 → (Phase 2 → Phase 3 반복)

학습 방침
프로젝트 진행 중 파이썬 문법과 개념을 C#과 비교하며 설명. 중간중간 이해도 확인 퀴즈 출제.
