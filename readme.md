# ChulAutoStock - 24시간 자동 주식 트레이딩 시스템

## 📌 프로젝트 개요
**목표**: 한국투자증권 API를 활용한 완전 자동 매매 시스템
- 24시간 상시 구동 (서버 재시작까지 무한 실행)
- 매일 08:28 자동 시작, 10:00 자동 종료
- 실시간 구글 시트 연동 (모바일 확인 가능)
- Slack 알림 시스템

## 🔄 Phase 시스템

### Phase 0: 인증 (08:29)
- API 토큰 발급 및 갱신
- 계좌 상태 확인

### Phase 1: 과거 데이터 수집 (08:30)
- KOSPI 100종목 전일 데이터 수집
- 기술적 지표 계산

### Phase 2: 실시간 모니터링 (08:35~08:57)
- 100종목 실시간 감시
- +2%~+4% 상승 종목 필터링
- 23분간 반복 실행

### Phase 3: 최종 종목 선정 (08:58)
- 상위 3종목 자동 선정
- 스코어링 기반 우선순위

### Phase 4: 포지션 관리 (09:00~09:59)
- **08:59**: 자동 매수 주문
- **09:00~09:58**: 실시간 모니터링
  - 익절: +4% 자동 매도
  - 손절: -2% 자동 매도
- **09:59**: 전량 청산

### Phase 5: 일일 마감 (09:59~10:00)
- 거래 리포트 생성
- 통계 저장

## 📱 실시간 확인
- **구글 시트**: 매매 내역 자동 기록
- **Slack**: 실시간 알림
- **모바일**: 구글 시트 앱에서 확인

## 🚀 실행 방법
```bash
# 24시간 자동 실행
python main.py

# 스케줄러 등록 (매일 08:28 자동 시작)
python reset_scheduler.py

# 스케줄러 삭제
python remove_scheduler.py
```

## ⚙️ 설정 파일
- `.env`: API 키, 계좌 정보
- `credentials.json`: 구글 시트 인증
- `AutoStockSetting.py`: 매매 조건 설정

## 📊 주요 기능
- ✅ 24시간 무인 운영
- ✅ 자동 매수/매도
- ✅ 실시간 익절/손절
- ✅ 구글 시트 기록
- ✅ Slack 알림
- ✅ 일일 리포트
